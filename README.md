# March7thAssistantExtender(三月七助手拓展包)
- 本程序作为[三月七助手(March7thAssistant)](https://github.com/moesnow/March7thAssistant)的拓展包
- 需要下载原版并在本程序的comfig.yaml里进行设置绑定路径
# 注意事项
- 需要手动调整游戏内分辨率为1920X1080
